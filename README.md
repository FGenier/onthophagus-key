# onthophagus-key

Clé interactive multilingue des groupes d’Onthophagus de Henri d’Orbigny (1913), version 1.2.

Version 1.2 :
- retour à l’accueil par clic sur le portrait de d’Orbigny dans l’en-tête ;
- boutons d’accueil « Reprendre l’identification » et « Nouvelle identification » ;
- conservation de la position lors du changement de langue, y compris dans un résultat/diagnose ;
- conservation du formatage du texte ajouté au 1er groupe.
